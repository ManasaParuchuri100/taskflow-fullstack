# TaskFlow API

A RESTful backend API built using Node.js and Express for managing tasks.

## Features

- Get all tasks
- Create tasks
- Update tasks
- Delete tasks
- Middleware validation
- Global error handling
- Environment variables

## Technologies

- Node.js
- Express.js
- CORS
- Morgan
- Helmet
- Dotenv

## Run Project

Install dependencies:

```bash
npm install
```

Start server:

```bash
npm run dev
```

Server:

http://localhost:5000
